## Stack
- Uses **LIFO**
Has operations:
- `pop()`
- `push()`
- `peek()`
- `isEmpty()`

## Queue
- Uses **FIFO**
Has operations:
- `add(item)`
- `remove()`
- `peek()`
- `isEmpty()`

For both:
- Finding the ith element takes `O(N)`
- Adding/Removing takes `O(1)`
